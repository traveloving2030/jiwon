# CSS 선언방법

```css
span { /*Selector*/
  color : red; /*Property, Value*/
}

```

1. Inline : HTML 태그 안에 스타일 속성넣기 (가장 우선순위가 높음)

```html
<p style="border:1px solid gray;color:red;font-size:2em;">
```

2. Internal : Style 태그로 지정

```html
<head>
<style>
p  {
  font-size : 2em;
  border:1px solid gray;
  color: red;
}
</style>
</head>

<body>
<div>...</div>
</body>
```

3. External : 외부파일(.css)로 지정

```html
<html>
	<head>
		<link rel="stylesheet" href="style.css">
	</head>
</html>
```
