# 머신러닝에 사용되는 Python 기본 문법 이해

## 넘파이(Numpy) : 파이썬의 확장 모듀르 간단한 코드로 데이터를 효율적ㅇ로 처리할 수 있다. 다차원 배열을 충실하게 지원

import numpy as np
a=np.array([0,1,2,3,4,5])
print(a)

- 결과 : [0 1 2 3 4 5]
